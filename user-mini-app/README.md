# 🚧 用户端小程序 (暂停开发)

> **注意：此目录下的微信小程序端目前已暂停开发。**

本目录原本计划作为游客、农户的轻量级移动端入口（基于 Uni-app / Vue3）。
但是在敏捷迭代中，我们优先完善了 Web 端的前台（包含响应式适配）以及基于 React+Vite 的管理端后台，大部分 C 端核心业务逻辑（如音视频播放、高德地图交互、智慧助农审批等）已经在 `Web-frontend` 和 `backend` 服务端联调跑通并完成集成开发。

因此，经过项目进度评估，这部分原生小程序代码库已打上标记暂停继续研发，留作未来可能的迁移备选方案。请评审人员或后续协作者以 `Web-frontend` 目录作为标准的用户前台体验入口。
